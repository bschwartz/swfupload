SWFUpload R4 Demos

FormsDemo:  Demonstrates using SWFUpload as part of a web form for submitting a job application.
	- Shows graceful degradation
	- Shows delayed file upload
	- Shows form submission after the upload queue is complete

MultiUploadDemo: Demonstrates how to include two seperate SWFUpload controls on a single page
	- Shows immediate uploading
	- Shows upload event handling
